## Getting Started 

In order to run the program, enter terminal and cd to the (../TOWER_OF_HANOI) directory.

For n amount of disks:

to run the Iterative solution type:
    ```
    dotnet run -Iterative n
    ```

to run the Recursive Solution type:
    ```
    dotnet run -Recursive n
    ```
